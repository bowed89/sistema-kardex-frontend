Card

Card is a section to group content and layout provides a built-in css for it. Apply .card style class to your container to use it. If the card has a title defined with h1 tag, add card-w-title to adjust paddings.

<div class="card">
    Content here
</div>

<div class="card card-w-title">
    <h1>Card with Title<h1>
    Content here
</div>

Input Animations

Label of an input can be animated on focus by wrapping both the input and label in an element with md-inputfield style class.


<span class="md-inputfield">
    <input type="text" pInputText>
    <label>Name</label>
</span>

Input fields can be filled as well by adding "md-inputfield-fill"

<span class="md-inputfield md-inputfield-fill">
    <input type="text" pInputText>
    <label>Name</label>
</span>

Textbox is an extension to md-inputfield to fill the label and inputfield together.

<span class="md-inputfield md-inputfield-box">
    <input type="text" pInputText>
    <label>Name</label>
</span>
